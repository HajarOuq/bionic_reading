# Bionic Reading Web App

A Django-based web application that converts text into **Bionic Reading format**, highlighting the first letters of words to improve reading speed and comprehension. 
Users can upload `.txt` or `.pdf` files, paste text directly, view processed text on the website, and download it as `.pdf`.

## Features

- **Text Input Options**
  - Upload `.txt` files
  - Upload `.pdf` files
  - Paste text directly into a textarea

- **Text Processing**
  - Convert uploaded or pasted text into Bionic Reading format
  - Handles different file types

- **Output Options**
  - View processed text directly on the website
  - Download processed text as `.pdf`.
